Hướng dẫn Cài đặt Code::Blocks

Phiên bản 25.03 (Bản chuẩn có Mingw)

Chào các bạn học sinh, để đảm bảo cả lớp chúng ta sử dụng chung một công cụ lập trình C++ (gọi là IDE), chúng ta sẽ cùng nhau cài đặt Code::Blocks. Đây là một IDE miễn phí, nhẹ nhàng và rất phù hợp cho người mới bắt đầu.

Phần 1: Tải về (Bước quan trọng nhất!)

Đây là bước dễ nhầm lẫn nhất. Các bạn phải tải đúng phiên bản có kèm theo "trình biên dịch" (compiler) - công cụ để dịch code của bạn sang ngôn ngữ máy tính.

Chú ý QUAN TRỌNG!

Phiên bản chúng ta cần tải là bản 25.03 VÀ có chữ mingw trong tên file. Tên chính xác là:

codeblocks-25.03mingw-setup.exe

CẢNH BÁO!

Tuyệt đối KHÔNG tải tệp chỉ có tên codeblocks-25.03-setup.exe (bản không có mingw). Nếu tải tệp này, bạn sẽ không thể biên dịch và chạy code được.

Nhấn vào đây để tải về

Link sẽ mở ra trang SourceForge và tự động tải về sau vài giây.

File bạn tải về sẽ có tên như thế này

Phần 2: Cài đặt chương trình

Sau khi tải xong tệp .exe, bạn hãy mở nó lên để cài đặt. Các bước cài đặt rất đơn giản:

  1. Welcome: Nhấn Next để tiếp tục.
  2. License Agreement (Điều khoản): Nhấn I Agree.
  3. Choose Components (Chọn thành phần): Cứ để mặc định là "Full". Nhấn Next.
  4. Install Location (Chọn nơi cài đặt): Bạn nên để đường dẫn mặc định. Nhấn Install.
  5. Chờ cho chương trình cài đặt xong.
  6. Một hộp thoại sẽ hiện ra hỏi: "Do you want to run Code::Blocks now?". Bạn có thể nhấn Yes để chạy luôn.

Phần 3: Thiết lập trình biên dịch (Lần chạy đầu tiên)

Khi bạn chạy Code::Blocks lần đầu tiên, nó sẽ tự động tìm trình biên dịch.

  1. Một cửa sổ "Compiler auto-detection" (Tự động phát hiện trình biên dịch) sẽ hiện lên.
  2. Nó sẽ tìm thấy "GNU GCC Compiler" và hiển thị trong danh sách.
  3. Bạn chỉ cần nhấn vào dòng đó, sau đó nhấn nút "Set as default" (Đặt làm mặc định).
  4. Nhấn OK.

Hình minh họa chọn trình biên dịch

Phần 4: Tạo và Chạy chương trình "Hello World"

Bây giờ chúng ta sẽ kiểm tra xem mọi thứ đã hoạt động đúng chưa.

  1. Vào menu File → New → Project...
  2. Chọn "Console application" và nhấn Go.
  3. Nhấn Next, sau đó chọn ngôn ngữ là C++.
  4. Project title: Gõ vào tên dự án, ví dụ: HelloWorld.
  5. Folder to create project in: Chọn một thư mục dễ nhớ để lưu code.
  6. Nhấn Next, sau đó nhấn Finish.

Dự án của bạn đã được tạo. Ở cửa sổ bên trái, bạn hãy mở rộng Sources và nháy đúp chuột vào tệp main.cpp.

Màn hình soạn thảo sẽ hiện ra đoạn code C++ mẫu:

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

Phần 5: Chạy chương trình

Trên thanh công cụ, bạn hãy tìm nút Build and Run (có hình bánh răng cưa và mũi tên tam giác màu xanh, phím tắt là F9).

Nút "Build and Run" quan trọng

Nhấn vào nút đó. Nếu mọi thứ chính xác, một cửa sổ console (màu đen) sẽ hiện lên và in ra dòng chữ:

Hello world!

🎉 Chúc mừng!

Bạn đã cài đặt thành công Code::Blocks và sẵn sàng để học C++!

Nếu gặp lỗi?

Lỗi "can't find compiler executable"?

Rất có thể bạn đã không tải đúng file, hoặc đã nhấn Cancel ở bước Phần 3. Hãy gỡ cài đặt Code::Blocks và làm lại, đảm bảo bạn tải đúng tệp mingw và chọn "GNU GCC Compiler" làm mặc định.
